This directory contains all Javascript files, including jQuery plugins.

Directory content:
------------------

 - jquery.min.js - Latest minified version of jQuery library. 
 - jquery.pngFix.pack.js - IE5.5+ PNG Alpha Fix. http://jquery.andreaseberhard.de/pngFix/index.html
 - iepngfix.htc - IE5.5+ PNG Alpha Fix v2.0 Alpha (use either this or jquery.pngFix.pack.js).
 - modernizr.min.js - detects support for browsers and enables HTML5 use. http://www.modernizr.com/
 - scripts.js - Main scripts file. Use this file for all custom scripting.


Useful jQuery plugins:
----------------------

 - jQuery UI - Theme-based jQuery User Interface plugins.
   http://jqueryui.com/download
 - Example - Show text in the input field and removes on focus. The text is restore on blur.
   http://mucur.name/system/jquery_example/

 
SOME TIPS:
----------

 - Consider using CDN for jQuery source (example page has a code for it) 
   http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js.
 - Use proper file naming within your projects.
   All jQuery plugins start with "jquery." and do not contain the version. 
   For example, "tabs-1.2.1.js" should be converted to "jquery.tabs.js"
 - Use suffix in file name if several custom scripting files required.
   For example, "scripts-front.css" or "scripts-main.css" etc.
 - Always minify your scripts before deployment.
