Plain Page - XHTML CSS JS Starter Kit

Project page: http://code.google.com/p/plainpage
License: LICENSE.TXT (http://www.gnu.org/licenses/old-licenses/gpl-2.0.html)

This directory contains framework-related helper images.

Please do not put any content-related images, like pictures or photos, in this directory.
Instead, use appropriate directory on your site (media, images etc).

There are  lots of different freeware icon packages available.
Here is the list of some of them:
- Iconza - collection of free icons that can be colored and reduced in size to your taste.
  http://iconza.com
- jQuery UI Icons - set of icons as a part jQuery UI project
  http://jqueryui.com/themeroller/
- Fugue Icons - 3,000 icons are included in PNG format (16x16 pixels) + bonus.
  http://p.yusukekamiyamane.com/
