This directory contains wireframes library for Adobe Fireworks.

This directory is updatable.
Please do not store any files in current directory.

Installation
------------
To install library, copy this directory into
c:\Users\%username%\AppData\Roaming\Adobe\Fireworks CS5\Common Library\ (Vista, 7)
or 
c:\Documents and Settings\%username%\Application Data\Roaming\Adobe\Fireworks CS5\Common Library\ (XP)

For example, for file Wireframes_Heading1.jsf, the path in Windows 7 will be 
c:\Users\%username%\AppData\Roaming\Adobe\Fireworks CS5\Common Library\Wireframes_Heading1.jsf

Restart Fireworks. The library will appear under Common Library pane.
