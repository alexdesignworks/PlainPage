This directory contains a set of grids with pre-defined guides to be used with Adobe Fireworks.
Also, this directory contains test grid backgrounds to be used with test html layouts (typography.html).

This directory is updatable.
Please do not store any files in current directory.

Grid file name format
---------------------
Grid_<width in px>_<column quantity>_<line height in px>.png

Zebra file name format (used for tests)
---------------------------------------
Zebra<height in px>.png

Blocks file name format (used for tests)
---------------------------------------
Block_<height in px>px_R<qty of rows>x<distance between columns in px>_C<qty of columns per width>x<column width in px>.png
